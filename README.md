# APEL-2025-2026-theatre
Affiche pour l'annonce du spectacle de fin d'année en partenariat avec l'APEL ainsi que les tickets d'entrée.

# Outils

Ordinateur sous Windows 11.

L'affiche et les tickets ont été créés avec Word de la suite office 365.

Le fond de l'affiche ainsi que l'image d'illustration sur les tickets d'entrée ont été créés par chatGPT et modifiés par gemini.

Famille de police utilisée : "Bodoni MT"

# Pour modifier l'affiche

Ouvrir le fichier source (APEL-2025-2026-theatre.docx) avec Word. Les polices sont standard dans word.

# Pour modifier les tickets d'entrée

Ouvrir le fichier source (APEL-2025-2026-ticket_theatre_orchestre.docx ou autre) avec Word. Les polices sont standard dans word.

# Suppléments

Une version image PNG et une version PDF sont également fournies.
